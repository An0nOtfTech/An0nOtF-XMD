## 🛠️ Manual Setup (VPS)

```bash
# 1. Clone the repo
git clone https://github.com/An0nOtFTech/An0nOtF-XMD
cd An0nOtF-XMD

# 2. Install dependencies
npm install

# 3. Start the bot
node index.js
```

On first start, visit the pairing panel to link your WhatsApp:
```
https://an0n0tf-xmd-pairing-server.up.railway.app
Or
https://an0n0tf-xmd-pairing-server.vercel.app 
```

## 🌐 Environment Variables

For cloud deployments (Heroku, Railway, Render) set these config vars:

| Variable | Description | Default | Example |
|---|---|---|---|
| `SESSION` | Session string from pairing panel | required | `=== AN0NOTF SESSION ===...` |
| `OWNER_NUMBER` | Your WhatsApp number with country code | required | `254712345678` |
| `BOT_NAME` | Custom bot name | `An0nOtF XMD` | `My Bot` |
| `PREFIX` | Command prefix | `!` | `!` or `.` or `/` |
| `MODE` | Bot mode | `public` | `public` / `private` / `pm` |
| `AUTO_TYPING` | Show typing animation before replies | `true` | `true` / `false` |
| `AUTO_REACT` | React to every command with emoji | `true` | `true` / `false` |
| `AUTO_VIEW_STATUS` | Auto view WhatsApp statuses | `true` | `true` / `false` |
| `AUTO_LIKE_STATUS` | Auto like WhatsApp statuses | `true` | `true` / `false` |
| `ALWAYS_ONLINE` | Keep bot always online 24/7 | `true` | `true` / `false` |
